1. download from apache foundation website;
2. set up $JAVA_HOME
	  $CATALINA_HOME

3. run sh catalina.sh start

2. Add CDT plugin for C/C++ project;

3. Add PyDev plugin for python project;
   repository address:http://pydev.org/updates
